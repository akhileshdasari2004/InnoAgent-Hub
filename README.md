## InnoAgent-Hub

**InnoAgent-Hub** is a multi-agent platform for open innovation, built on Coral Protocol.

### Quick Start (Root)

1. **Check dependencies**

```bash
./scripts/check-dependencies.sh
```

2. **Start Coral Server**

```bash
./scripts/start-server.sh
```

3. **Start Coral Studio**

```bash
cd frontend/studio
./start-studio.sh
```
